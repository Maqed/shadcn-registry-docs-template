## Reference

### Props

The `Button` component accepts all props from [`@base-ui-components/react` Button](https://base-ui.com/react/components/button) plus the following:

| Prop      | Type                                                                       | Default     |
| --------- | -------------------------------------------------------------------------- | ----------- |
| `variant` | `"default" \| "secondary" \| "outline" \| "ghost" \| "destructive" \| "link"` | `"default"` |
| `size`    | `"default" \| "xs" \| "sm" \| "lg" \| "icon" \| "icon-xs" \| "icon-sm" \| "icon-lg"` | `"default"` |
